## 🧩 刷题任务

### 题目：二进制差异数（200分）

**题目描述**：
对于任意两个正整数 A 和 B，定义它们的差异值：差异值 = A 与 B 的二进制按位异或结果中 1 的个数。

定义：对于任意两个正整数 A 和 B，如果 `A & B == 0`，且 `A | B` 的二进制表示中 1 的个数记为 k，则称 A 和 B 的差异值为 k。

给定一个正整数数组，求其中有多少对 (i, j) 满足 i < j 且 A[i] 与 A[j] 的差异值为 A[i] | A[j] 的二进制 1 的个数。

**即**：求满足 `A[i] & A[j] == 0` 的数对个数。

**输入描述**：
第一行：正整数 N（2 ≤ N ≤ 10^5）
第二行：N 个正整数（范围 [1, 2^20]）

**输出描述**：
满足条件的数据对个数

**样例输入**：
```
5
3 4 2 1 5
```

**样例输出**：
```
5
```

**解释**：
满足 `A[i] & A[j] == 0` 的数对有：
(3,4): 011 & 100 = 000 ✓
(3,2): 011 & 010 = 010 ≠ 0 ✗
(3,1): 011 & 001 = 001 ≠ 0 ✗
(3,5): 011 & 101 = 001 ≠ 0 ✗
(4,2): 100 & 010 = 000 ✓
(4,1): 100 & 001 = 000 ✓
(4,5): 100 & 101 = 100 ≠ 0 ✗
(2,1): 010 & 001 = 000 ✓
(2,5): 010 & 101 = 000 ✓
(1,5): 001 & 101 = 001 ≠ 0 ✗
共5对。

---

**思路分析**：

**关键洞察**：
本题条件 `A & B == 0` 等价于：A 和 B 的二进制表示中，没有任何一位同时为 1。

**朴素解法 O(N²)**：两两判断 → N=10^5 时会超时。

**优化思路**：
1. 由于数值范围 ≤ 2^20，可以枚举每个数的所有 **子集补数**。
2. 对于每个数 x，它的补集（取反）中所有与 x 无交集（即 `x & y == 0`）的数 y，一定是 x 的某个子集的补数。
3. 具体：对于每个 x，枚举所有可能的 y（0 到 2^20-1），判断 x & y == 0，但这样 O(N×2^20) 更大。

**更优方法——按最高位分组**：
- 如果 `x & y == 0`，那么 x 的最高位 1 对应的位置，y 在那一位一定是 0。
- 利用 SOS DP（子集DP，Sum Over Subsets）统计每个数的「子集」出现次数。

**更简洁的常用解法（华为OD常见套路）**：
- 用哈希表统计每个数的出现次数。
- 对每个数 x，枚举它的所有子集（通过 `sub = (sub-1) & x` 的经典枚举子集技巧），然后统计互补部分。
- 实际上这里有个更巧妙的性质：`x & y == 0` 意味着 y 是 `~x` 的子集（在有效位范围内）。

**推荐实现**：
对每个数 x，枚举其补集（~x）的所有子集，看这些子集在数组中的出现次数。注意去重 (i, j) 时 i < j。

---

**参考代码**：
```python
def count_pairs(arr):
    from collections import Counter
    cnt = Counter(arr)
    nums = list(cnt.keys())
    MAX_BIT = 20  # 因为数值 ≤ 2^20
    ans = 0

    # 方法：对每个数 x，枚举其补集的所有子集
    for x in nums:
        # x 的补集（只考虑低20位）
        complement = ((1 << MAX_BIT) - 1) ^ x
        # 枚举 complement 的所有子集
        sub = complement
        while True:
            y = sub
            if y in cnt and y != x:
                ans += cnt[x] * cnt[y]
            elif y == x:
                # 自身与自身，需要 C(cnt[x], 2)
                pass  # 单独处理
            if sub == 0:
                break
            sub = (sub - 1) & complement

    # 处理 x == y 的情况：x & x == 0 只有当 x == 0，但题目说正整数，所以不可能
    # 但上面的枚举会重复计算 (x,y) 和 (y,x)，所以 ans 要除以 2
    ans //= 2

    # 处理 x == y 情况：对于 x & x == 0？只有当 x == 0 时成立，但范围 [1, 2^20]，跳过

    return ans

# 更简洁的写法（推荐）：直接两两枚举补集子集，哈希去重
def count_pairs_v2(arr):
    n = len(arr)
    cnt = {}
    for v in arr:
        cnt[v] = cnt.get(v, 0) + 1

    keys = list(cnt.keys())
    total = 0
    # 枚举所有不相同的数对
    for i in range(len(keys)):
        x = keys[i]
        # 枚举 x 的补集子集来配对
        comp = ((1 << 20) - 1) ^ x
        sub = comp
        while True:
            y = sub
            if y in cnt and y > x:  # 保证只统计一次
                total += cnt[x] * cnt[y]
            if sub == 0:
                break
            sub = (sub - 1) & comp

    # 处理相同数 x & x == 0? 只有 x=0 成立，忽略
    return total

# 输入处理
N = int(input())
arr = list(map(int, input().split()))
print(count_pairs_v2(arr))
```

**OD备考提示**：
- **位运算题**在 200分题目中频繁出现，核心是二进制性质。务必熟练掌握补码、子集枚举、SOS DP。
- **复杂度要求**：N 可达 10^5，必须 O(N × 子集数)，平均每个数的子集数约 2^(popcount(补数))，最坏情况是 2^20，但实际平均较小。
- **Java/C++ 考生注意**：Python 枚举子集在 2^20 时可能较慢，可加 `MAX_BIT=20` 限制。
- **查重去重**：枚举子集时会重复统计 (x,y) 和 (y,x)，务必用 `y > x` 条件或最后除以 2 去重。
- **边界情况**：所有数都相同的情况、数值达到 2^20 边界的情况。
