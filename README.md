# 🎯 华为OD机考 · 2个月备考计划

> 基于 LeetCode 75 经典题 + 华为OD历史真题的系统化备考方案

## 总览

| 阶段 | 时间 | 内容 | 目标 |
|------|------|------|------|
| **Phase 1** | 第1-4周 | LeetCode 75 经典题 | 夯实数据结构与算法基础 |
| **Phase 2** | 第5-8周 | 华为OD历史真题 | 熟悉考题风格，冲刺提分 |

## 每日结构

每天的内容包含以下三部分：

```
📖 知识点讲解 —— 当日核心概念 & 解题模板
🧩 刷题任务     —— 1-2道精选题目 + 思路分析
✅ 参考代码     —— Python 实现（含详细注释）
```

## 如何使用

- **在线阅读**：直接在 GitHub 上浏览各周目录
- **打印**：下载仓库 ZIP 或 clone 到本地打印
- **推荐节奏**：每天花 1-2 小时，先学知识点再做题
