## 📌 每天怎么学

每天花 **1-2小时**，按这个节奏：

| 步骤 | 时间 | 内容 |
|------|:----:|------|
| 📖 **学知识点** | 20min | 阅读当日知识点讲解，理解算法模板 |
| 🧩 **做题** | 40-60min | 完成1-2道刷题任务，先自己思考再看答案 |
| ✅ **对照代码** | 10min | 对比自己的思路和参考答案，理解**为什么** |
| 📝 **复盘** | 5min | 记下易错点、技巧（周末复习用） |

> 💡 **建议**：不要死磕太久——想10分钟没思路就直接看题解，重点学解题套路和模板。
