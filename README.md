# 🎯 华为OD机考 · 2个月高强度备考计划

> 融合 **LeetCode 75 + Top Interview 150 + 剑指Offer专项突破 + 华为OD历史真题**
> 每日 **6-7题**，8周冲刺

---

## 📌 每天怎么学（每天约2小时）

| 步骤 | 时间 | 内容 |
|:----|:----:|:------|
| 📖 **知识点速览** | 15min | 看当日主题的算法模板和核心套路 |
| 🧩 **刷题 × 6** | 75min | 平均12min/题，难易搭配，先自己思考再对答案 |
| ✅ **对照代码** | 15min | 重点看没做出来/做错的题，理解**为什么** |
| 📝 **错题记录** | 5min | 记下易错点和套路（周末复习用） |

> 💡 **策略**：简单题（⭐）快速过，中等题（⭐⭐）重点练，难题（⭐⭐⭐）看思路即可，不硬磕
