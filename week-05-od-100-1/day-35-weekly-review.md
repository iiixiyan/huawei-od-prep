### 二、高频易错点复盘

#### 1. 字符串处理
- **双指针法**是回文、子串类问题的基础，注意边界条件
- `.strip()` 清理输入中的换行和空格
- 字符判断用 `.isalpha()`、`.isdigit()`、`.isalnum()`

#### 2. 数组操作
- **滑动窗口**解决连续子数组和/积问题，复杂度 O(n)
- 窗口满足条件后的计数技巧：`count += n - right`
- 注意数据范围，大数组用 O(n) 算法

#### 3. 排序相关
- Python 稳定排序（Timsort）可分解多关键字排序
- 排序 key 元组写法：`(-freq, first_pos)` 实现「降序、升序」
- 首次出现位置需显式记录，不依赖字典顺序

#### 4. 模拟题
- 仔细读题，把规则逐一翻译成代码
- 贪心策略在 OD 模拟题中常适用，但需验证
- 向上取整公式：`(a + b - 1) // b`

#### 5. 通用技巧
- 输入解析：`map(int, input().split())`
- defaultdict / Counter 简化哈希计数
- 列表模拟栈/队列操作

### 三、限时小测（40分钟，4道题）

#### 第1题：反转句子单词
**题目**：给定一个英文句子（单词间用空格分隔），将每个单词反转，但保持单词在句子中的顺序不变。输出反转后的句子。

**输入**：
```
Hello World
```
**输出**：
```
olleH dlroW
```

**参考代码**（写在纸上/IDE中再对照）：
```python
s = input().strip()
words = s.split()
reversed_words = [w[::-1] for w in words]
print(' '.join(reversed_words))
```

#### 第2题：数组中位数
**题目**：给定 n 个整数，输出这组数的中位数（如果 n 为奇数，中位数为中间的数；如果 n 为偶数，中位数为中间两个数的平均值，向下取整）。

**输入**：
```
5
3 1 4 1 5
```
**输出**：
```
3
```

**参考代码**：
```python
n = int(input())
arr = list(map(int, input().split()))
arr.sort()
if n % 2 == 1:
    print(arr[n // 2])
else:
    print((arr[n // 2 - 1] + arr[n // 2]) // 2)
```

#### 第3题：连续字符压缩
**题目**：给定一个字符串，将连续重复的字符压缩为「字符+重复次数」的形式。如果字符只出现一次，不输出次数。

**输入**：
```
aabcccccaaa
```
**输出**：
```
a2bc5a3
```

**参考代码**：
```python
s = input().strip()
res = []
count = 1
for i in range(1, len(s)):
    if s[i] == s[i - 1]:
        count += 1
    else:
        res.append(s[i - 1])
        if count > 1:
            res.append(str(count))
        count = 1
res.append(s[-1])
if count > 1:
    res.append(str(count))
print(''.join(res))
```

#### 第4题：最小公倍数
**题目**：给定两个正整数 a 和 b，求它们的最小公倍数（LCM）。

**输入**：
```
12 18
```
**输出**：
```
36
```

**参考代码**：
```python
import math
a, b = map(int, input().split())
print(a * b // math.gcd(a, b))
```
