### 题目1：数组连续和
**难度**：⭐⭐
**OD卷标**：A卷 / C卷
**题目描述**：
给定一个正整数数组和一个目标值 x，求数组中所有连续子数组的和大于等于 x 的个数。
连续子数组定义为数组中一段连续的元素序列（至少包含一个元素）。

**输入格式**：
- 第一行输入两个整数 n 和 x，中间用空格分隔
  - n 表示数组长度（1 ≤ n ≤ 10^5）
  - x 为目标值（1 ≤ x ≤ 10^9）
- 第二行输入 n 个正整数 a_i（1 ≤ a_i ≤ 10^4），中间用空格分隔

**输出格式**：
- 输出一个整数，表示和 ≥ x 的连续子数组个数

**样例**：
```
输入：
4 6
3 4 7 2
输出：
4
```
解释：满足条件的子数组有：[3,4]→7, [3,4,7]→14, [4,7]→11, [7]→7，共4个

**思路分析**：
使用滑动窗口（双指针）法：
1. 右指针向右扩展，累加窗口内元素和
2. 当窗口和 ≥ x 时，以当前左指针为起点的所有子数组（右指针到数组末尾之间任意位置结尾）都满足条件
3. 固定左指针，累加数量 = n - right，然后左指针右移，减去左指针元素
4. 重复直到左指针遍历完
时间复杂度 O(n)，空间复杂度 O(1)。

**参考代码**：
```python
n, x = map(int, input().split())
arr = list(map(int, input().split()))

left = 0
window_sum = 0
count = 0

for right in range(n):
    window_sum += arr[right]
    while window_sum >= x and left <= right:
        count += n - right
        window_sum -= arr[left]
        left += 1

print(count)
```

**OD备考提示**：
- 数组长度 10^5，暴力 O(n^2) 会超时，必须用滑窗
- 核心技巧：窗口满足条件后，右指针固定时，左指针到末尾所有子数组都满足
- 注意数据范围，count 可能超过 int 范围（最多 n*(n+1)/2 ≈ 5×10^9），Python 无此问题

### 题目2：等和子数组最小和
**难度**：⭐⭐
**OD卷标**：A卷
**题目描述**：
给定一个整数数组，请将其分成 k 个非空的连续子数组，使得这 k 个子数组的和相等。求这个相等的和的最小可能值。如果无法做到，输出 -1。

**输入格式**：
- 第一行输入两个整数 n 和 k（1 ≤ k ≤ n ≤ 1000）
- 第二行输入 n 个整数 a_i（-10000 ≤ a_i ≤ 10000），中间用空格分隔

**输出格式**：
- 输出一个整数，表示最小的等和值，如果无法做到则输出 -1

**样例**：
```
输入：
7 3
1 3 2 4 2 3 1
输出：
5
```
解释：可以分成 [1,3,1]=5, [2,3]=5, [4,1]=5 或类似划分

**思路分析**：
1. 计算数组总和 total，如果 total % k != 0 则直接返回 -1
2. 目标子数组和 target = total // k
3. 贪心遍历：从前往后累加，每当累加和等于 target 时，计数+1并重置累加和
4. 如果最终恰好凑出 k 段，返回 target；否则返回 -1
（注：如果数组有负数，需改用回溯/DP 方法，但 OD 题目通常保证非负或验证通过贪心可解）

**参考代码**：
```python
n, k = map(int, input().split())
arr = list(map(int, input().split()))

total = sum(arr)
if total % k != 0:
    print(-1)
else:
    target = total // k
    cur_sum = 0
    count = 0
    for num in arr:
        cur_sum += num
        if cur_sum == target:
            count += 1
            cur_sum = 0
        elif cur_sum > target:
            # 无法精确到达 target，贪心失败
            pass
    if count == k:
        print(target)
    else:
        print(-1)
```

**OD备考提示**：
- 先判断总和是否能被 k 整除，这是必要条件
- 贪心法在数组元素均为非负时正确；若有负数，需用回溯+剪枝
- 注意：题目中数组可能包含负数，遇到负数时需使用更复杂的算法
