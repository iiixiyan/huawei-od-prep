### 题目1：字符串重新排列、字符串重新排序
**难度**：⭐⭐
**OD卷标**：A卷
**题目描述**：
给定一个字符串，包含多个由空格分隔的单词。请对单词进行重新排序：
1. 按单词长度升序排列（短在前）
2. 长度相同时，按单词的字典序降序排列（字典序大的在前）
3. 如果两个单词完全相同，按出现顺序保留（稳定排序）

**输入格式**：
- 一行字符串，包含由空格分隔的若干单词，单词仅含字母
- 单词个数 ≤ 100，每个单词长度 ≤ 100

**输出格式**：
- 输出一行，按规则排序后的单词，用空格分隔

**样例**：
```
输入：
I am a student
输出：
a I am student
```
解释：a(1) < I(1) < am(2) < student(7)；长度1中有a和I，字典序降序I在前a在后，但这里按题目描述应该是…注意核对题目

正确排序说明：长度1的单词有 "I" 和 "a"，长度相同按字典序降序："a" > "I"（字典序比较），所以先 a 后 I？

实际上 "a" 的 ASCII 97 > "I" 的 ASCII 73，字典序降序时 a 在前。所以输出 "a I am student"。

**思路分析**：
1. 将字符串按空格分割为单词列表
2. 使用稳定排序：先按字典序降序排（长度相同时），再按长度升序排
3. Python 的 sorted() 是稳定的，可以用元组排序实现

**参考代码**：
```python
s = input().strip()
words = s.split()

# 稳定排序：先按字典序降序，再按长度升序
# 由于Python排序稳定，可以分两步
words.sort(key=lambda w: w, reverse=True)   # 字典序降序
words.sort(key=len)                          # 长度升序（稳定，保持同长度字典序降序）

print(' '.join(words))
```

**OD备考提示**：
- Python 的 Timsort 是稳定排序，可以利用这点分步排序
- 注意题目要求的排序优先级：长度升序 > 字典序降序
- 如果要求完全相同的单词保留原顺序，稳定排序自动保证

### 题目2：数组去重和排序
**难度**：⭐⭐
**OD卷标**：B卷
**题目描述**：
给定一个整数数组，请按照以下规则进行去重和排序：
1. 去除数组中重复的元素，每个元素只保留一个
2. 按元素在原数组中出现的次数降序排列
3. 出现次数相同时，按元素第一次出现的顺序排列（保持相对顺序）

**输入格式**：
- 第一行输入一个整数 n（1 ≤ n ≤ 1000）
- 第二行输入 n 个整数 a_i（-1000 ≤ a_i ≤ 1000），中间用空格分隔

**输出格式**：
- 输出一行，去重排序后的整数，用空格分隔

**样例**：
```
输入：
8
3 1 2 3 2 3 4 5
输出：
3 2 1 4 5
```
解释：3出现3次→第1，2出现2次→第2，1、4、5各1次→按首次出现顺序：1→4→5

**思路分析**：
1. 使用字典统计每个数字出现的次数，并记录首次出现的位置
2. 排序规则：次数降序 → 首次出现位置升序
3. 可以用 `sorted()` 的 key 函数实现

**参考代码**：
```python
n = int(input())
arr = list(map(int, input().split()))

freq = {}
first_pos = {}

for i, num in enumerate(arr):
    if num not in first_pos:
        first_pos[num] = i
    freq[num] = freq.get(num, 0) + 1

# 去重：获取所有不重复的数字
unique_nums = list(freq.keys())

# 排序：按频次降序，频次相同按首次出现位置升序
unique_nums.sort(key=lambda x: (-freq[x], first_pos[x]))

print(' '.join(map(str, unique_nums)))
```

**OD备考提示**：
- 去重后每个数字只输出一次，不是按出现次数输出多次
- 排序的 key 元组技巧：`(-次数, 首次位置)` 实现「次数降序、位置升序」
- Python 的 dict 在 3.7+ 保持插入顺序，但本题需显式记录 first_pos
