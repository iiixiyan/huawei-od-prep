### 题目1：模拟商场优惠打折
**难度**：⭐⭐
**OD卷标**：A卷
**题目描述**：
某商场有 n 件商品，每件商品有原价和折扣价（优惠后价格）。小明有 m 张优惠券，每张优惠券可用来减免一件商品的价格（在原价基础上打折），但每件商品最多只能用一张优惠券。

优惠券有两种类型：
- 类型1：满减券（满 X 减 Y），即商品原价 ≥ X 时，减免 Y 元
- 类型2：折扣券（打 Z% 折扣），即商品价格变为原价的 Z%，向上取整

小明想按顺序购买商品，每件商品都买且只买一次。小明可以自行选择每件商品是否使用优惠券以及使用哪种优惠券，但优惠券数量有限。请计算小明最少需要支付的总金额。

**输入格式**：
- 第一行输入三个整数 n, m1, m2，分别表示商品数量、满减券数量、折扣券数量
- 第二行输入 n 个整数，表示每件商品的原价
- 第三行输入两个整数 X, Y，表示满减券的规则（满 X 减 Y）

**输出格式**：
- 输出一个整数，表示最少支付金额

**样例**：
```
输入：
3 2 1
100 200 300
100 20
```
折扣券规则打8折（Z=80）。
使用策略：商品100→用满减券（满100减20→80元），商品200→用折扣券（200×80%=160元），商品300→用满减券（满100减20→280元）
总金额 = 80 + 160 + 280 = 520

**思路分析**：
1. 对每件商品计算三种价格：原价、满减价、折扣价
2. 由于 m1, m2 数量有限，需要合理分配优惠券使总价最小
3. 贪心策略：将所有商品的优惠力度排序，优先使用优惠券在优惠最大的商品上
4. 具体：计算每件商品用满减节省的钱和用折扣节省的钱，按节省额降序分配

**参考代码**：
```python
n, m1, m2 = map(int, input().split())
prices = list(map(int, input().split()))
X, Y = map(int, input().split())
Z = 80  # 折扣券折扣率

# 计算满减后的价格和折扣后的价格
discounts = []
for price in prices:
    manjian = price - Y if price >= X else price
    dazhe = (price * Z + 99) // 100  # 向上取整
    discounts.append((price, manjian, dazhe))

# 按能节省的金额排序
# 对每件商品，计算用满减节省的金额和用折扣节省的金额
savings = []
for price, mj, dz in discounts:
    save_mj = price - mj
    save_dz = price - dz
    savings.append((save_mj, save_dz, price, mj, dz))

# 节省金额降序排列
savings.sort(key=lambda x: max(x[0], x[1]), reverse=True)

total = 0
used_m1, used_m2 = 0, 0
for save_mj, save_dz, price, mj, dz in savings:
    # 优先使用节省更多的券
    if save_mj >= save_dz and used_m1 < m1:
        total += mj
        used_m1 += 1
    elif save_dz > save_mj and used_m2 < m2:
        total += dz
        used_m2 += 1
    elif used_m1 < m1:
        total += mj
        used_m1 += 1
    elif used_m2 < m2:
        total += dz
        used_m2 += 1
    else:
        total += price

print(total)
```

**OD备考提示**：
- 模拟题要仔细读题，理清优惠券的使用规则
- 金额计算注意向上取整：`(price * Z + 99) // 100`
- 贪心策略需要验证，OD 模拟题贪心通常正确

### 题目2：小明减肥
**难度**：⭐⭐
**OD卷标**：2025B
**题目描述**：
小明决定减肥，他计划在 n 天内每天走一定的步数。第一天走 1 步，之后每天的步数按以下规则确定：
- 如果前一天走的步数是奇数，则今天走「前一天的步数 × 3 + 1」步
- 如果前一天走的步数是偶数，则今天走「前一天的步数 ÷ 2」步

请计算第 n 天走的步数。

**输入格式**：
- 一行输入一个正整数 n（1 ≤ n ≤ 1000）

**输出格式**：
- 输出一个整数，表示第 n 天的步数

**样例**：
```
输入：
3
输出：
2
```
解释：第1天=1步(奇)→第2天=3×1+1=4步(偶)→第3天=4÷2=2步

**思路分析**：
1. 直接模拟每天的步数计算，按规则迭代
2. 注意 n 最大为 1000，步数可能很大（类似Collatz猜想），需要用 Python 的大整数

**参考代码**：
```python
n = int(input())
steps = 1  # 第一天

for day in range(2, n + 1):
    if steps % 2 == 1:  # 奇数
        steps = steps * 3 + 1
    else:               # 偶数
        steps = steps // 2

print(steps)
```

**OD备考提示**：
- 简单的过程模拟，直接按规则循环即可
- 注意 n=1 时直接输出 1（循环不执行）
- 步数可能非常大，Python 原生支持大整数无需担心溢出
