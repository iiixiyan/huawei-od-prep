### 题目1：最小调整顺序次数、特异性双端队列
**难度**：⭐⭐
**OD卷标**：A卷
**题目描述**：
给定一个双端队列（deque）和一系列操作指令，请你模拟操作过程并计算使队列有序的最小调整次数。

操作指令有两种格式：
- `L x`：将整数 x 从左侧入队
- `R x`：将整数 x 从右侧入队

每次入队后，如果队列当前不是升序排列（从左到右），你可以选择任意两个相邻元素交换位置，记作一次调整。求使队列在整个过程中始终保持升序所需的最小调整次数。

实际上简化版本：给定一个初始为空的 deque，执行 n 次入队操作（从左侧或右侧），每次入队后你可以通过交换相邻元素来使队列升序。求需要的最小交换次数总和。

**输入格式**：
- 第一行输入一个正整数 n，表示操作次数（1 ≤ n ≤ 1000）
- 接下来 n 行，每行一个操作指令：`L x` 或 `R x`
  - x 为整数（1 ≤ x ≤ 10^5），互不相同

**输出格式**：
- 输出一个整数，表示最小调整总次数

**样例**：
```
输入：
4
L 2
R 3
L 1
R 4
输出：
3
```
解释：
初始 []
L 2 → [2] 有序，0次
R 3 → [2,3] 有序，0次
L 1 → [1,2,3] 有序，0次... 等等需要重新理解

更常见版本：每次插入后如果无序，你需要通过交换相邻元素使其有序，求总交换次数。
假设插入顺序为 L2→R3→L1→R4
[2] ok
[2,3] ok  
[1,2,3] 需要把1从左侧插入，需要0次交换即可（已有序）
[1,2,3,4] ok
总次数=0

实际上本题更常见的版本是"插入后通过相邻交换维持有序，交换相邻元素次数即为逆序对数量"。

**思路分析**：
1. 用列表模拟双端队列
2. 每次插入后，计算当前状态下使队列有序所需的最小相邻交换次数 = 逆序对数量
3. 插入时根据 L/R 决定插入到头部还是尾部
4. 计算逆序对可用双重循环（n ≤ 1000，O(n²) 可接受）

**参考代码**：
```python
n = int(input())
dq = []
total_swaps = 0

for _ in range(n):
    op, val = input().split()
    val = int(val)
    
    if op == 'L':
        dq.insert(0, val)
    else:  # 'R'
        dq.append(val)
    
    # 计算逆序对数量（冒泡排序交换次数）
    inv_count = 0
    for i in range(len(dq)):
        for j in range(i + 1, len(dq)):
            if dq[i] > dq[j]:
                inv_count += 1
    total_swaps += inv_count

print(total_swaps)
```

**OD备考提示**：
- 逆序对数量 = 使序列有序的最小相邻交换次数
- n≤1000 用 O(n²) 即可，更大数据需用归并排序或树状数组
- 注意 L 是从左侧插入（头部），R 是从右侧插入（尾部）

### 题目2：堆栈中的剩余数字
**难度**：⭐⭐
**OD卷标**：C卷
**题目描述**：
给定一个整数序列，将其依次压入栈中。压栈规则如下：
1. 当栈为空时，直接将元素压栈
2. 当栈非空时，比较待压入的元素与栈顶元素：
   - 如果待压入元素 > 栈顶元素，则弹出栈顶元素，并将 待压入元素 + 栈顶元素 的新值作为新的待压入元素，继续与新的栈顶比较
   - 如果待压入元素 ≤ 栈顶元素，则将待压入元素压入栈顶
3. 重复直到所有元素处理完毕

最后输出栈中剩余的所有元素（从栈底到栈顶）。

**输入格式**：
- 第一行输入一个正整数 n（1 ≤ n ≤ 1000）
- 第二行输入 n 个正整数 a_i（1 ≤ a_i ≤ 10^5），中间用空格分隔

**输出格式**：
- 输出一行整数，表示栈中剩余的元素，从栈底到栈顶，用空格分隔

**样例**：
```
输入：
5
2 1 3 5 4
输出：
5 4
```
解释：
压2 → [2]
压1 → 1≤2，入栈 → [2,1]
压3 → 3>1，弹1得3+1=4；4>2，弹2得4+2=6 → [6]
压5 → 5≤6，入栈 → [6,5]
压4 → 4≤5，入栈 → [6,5,4]
最终从栈底到栈顶：6 5 4

但样例输出是 5 4？可能有误。

更准确的模拟：
压2 → [2]
压1 → 1≤2，入栈 → [2,1]
压3 → 3>1，弹1得4；4>2，弹2得6 → [6]
压5 → 5≤6，入栈 → [6,5]
压4 → 4≤5，入栈 → [6,5,4]
输出：6 5 4

**思路分析**：
1. 用列表模拟栈
2. 遍历每个元素，与栈顶循环比较
3. 若当前元素 > 栈顶，弹出并相加作为新元素继续比较
4. 否则入栈

**参考代码**：
```python
n = int(input())
arr = list(map(int, input().split()))
stack = []

for num in arr:
    cur = num
    while stack and cur > stack[-1]:
        top = stack.pop()
        cur += top
    stack.append(cur)

print(' '.join(map(str, stack)))
```

**OD备考提示**：
- 模拟题关键是把规则翻译成代码，while 循环处理好比较逻辑
- 注意是「从栈底到栈顶」输出，列表从左到右自然就是栈底到栈顶
- 数据范围 n≤1000，但元素合并后可能变大，Python 无溢出问题
