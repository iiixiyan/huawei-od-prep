## Problem 1: 任务调度器 (100分)

### 题目描述

给定一个任务数组 `tasks`，其中 `tasks[i]` 表示第 i 个任务需要占用的CPU时间片数（单位为毫秒）。系统有一个"冷却时间"约束：**同一种类型的任务执行后必须等待至少 n 毫秒才能再次执行**。任务类型用大写字母 `'A'-'Z'` 表示。

你的任务是最小化完成任务所需的总时间（时间单位：毫秒）。任务可以在任意顺序执行，且每个时间单位可以执行一个任务（占用对应毫秒数）。

> 注意：本题中每个任务的时间片数相同，均为1单位时间。即每个任务执行需要1毫秒。

### 输入格式

两行：
- 第一行：字符串 `tasks`，包含大写字母，相邻任务类型无分隔符
- 第二行：整数 `n`，表示同类型任务的冷却时间

### 输出格式

一个整数，表示完成所有任务所需的最少时间单位数。

### 样例

```
输入:
AAABBB
2

输出:
8

解释:
A -> B -> idle -> A -> B -> idle -> A -> B
需要 8 个单位时间
```

```
输入:
ACCCEEE
2

输出:
7

解释:
C -> E -> A -> C -> E -> C -> E
需要 7 个单位时间
```

### 数据范围

| 参数 | 范围 |
|------|------|
| tasks 长度 | 1 ≤ len ≤ 10⁴ |
| n | 0 ≤ n ≤ 100 |
| 任务类型 | 仅大写字母 A-Z |

### 解题思路

**贪心 + 数学推导**：

1. 统计每种任务的出现次数，记最多出现次数为 `max_count`
2. 出现次数为 `max_count` 的任务类型数为 `max_types`
3. 理论最少时间 = `(max_count - 1) * (n + 1) + max_types`
4. 但结果不能小于总任务数（当冷却时间 n 很小时）

**核心公式**：
```
result = max(len(tasks), (max_count - 1) * (n + 1) + max_types)
```

### 参考代码

```python
from collections import Counter

def least_interval(tasks, n):
    # 统计频率
    freq = Counter(tasks)
    max_count = max(freq.values())
    # 有多少种任务达到了最大频率
    max_types = sum(1 for v in freq.values() if v == max_count)
    
    # 公式计算
    result = (max_count - 1) * (n + 1) + max_types
    
    # 结果不能小于总任务数
    return max(len(tasks), result)


if __name__ == "__main__":
    tasks = input().strip()
    n = int(input().strip())
    print(least_interval(tasks, n))
```

## Problem 2: 迷宫最短路径 (200分)

### 题目描述

给定一个 `N × M` 的二维迷宫，包含以下元素：
- `S`：起点
- `E`：终点
- `0`：可通行空地
- `1`：障碍物（不可通行）
- `A` ~ `J`：钥匙（共10种，分别对应 `a` ~ `j` 锁）
- `a` ~ `j`：锁（需要对应钥匙才能通过，如 `a` 需要 `A` 钥匙）

**规则：**
1. 每次可以向上、下、左、右四个方向移动一格
2. 不能走出边界
3. 不能进入障碍物 `1`
4. 遇到锁 `a`~`j`，如果没有对应钥匙则无法进入
5. 经过钥匙点时自动拾取，钥匙携带数量不限
6. 求从 `S` 到 `E` 的**最短路径长度**（步数），如果不可达输出 `-1`

### 输入格式

第一行：两个整数 `N` 和 `M`（1 ≤ N, M ≤ 100）  
接下来 N 行：每行一个长度为 M 的字符串，表示迷宫

### 输出格式

一个整数，表示最短路径长度。如果不可达，输出 `-1`。

### 样例

```
输入:
5 5
S0001
10a0E
0A000
00000
00000

输出:
10

解释:
S(0,0) -> →(0,1) →(0,2) →(0,3) →(0,4) 被1挡住
实际路径：S→右→下→下→下→右→右→右...(需取钥匙A)
```

```
输入:
3 3
S1E
101
0a0

输出:
-1

解释: 终点被锁挡住且无法拿到钥匙
```

### 数据范围

| 参数 | 范围 |
|------|------|
| N, M | 1 ≤ N, M ≤ 100 |
| 网格大小 | N*M ≤ 10000 |
| 钥匙类型 | A~J，最多10种 |
| 钥匙数量 | 每种至少1把，最多不限 |

### 解题思路

**状态压缩BFS**（经典"钥匙与锁"问题）：

1. **状态定义**：`(x, y, key_mask)` 表示在位置 `(x, y)` 且已拥有的钥匙集合为 `key_mask`
2. **key_mask**：使用10位二进制数表示钥匙拥有情况
   - 第 0 位 = 1 表示有钥匙 A
   - 第 1 位 = 2 表示有钥匙 B
   - 以此类推
3. **BFS过程**：
   - 从起点 `(S_x, S_y, 0)` 开始
   - 遇到钥匙（大写字母）：更新 `key_mask` 对应位为1
   - 遇到锁（小写字母）：检查 `key_mask` 对应位是否为1，为1才能进入
   - 遇到终点 E：返回当前步数
4. **距离数组**：`dist[N][M][1<<10]`，初始化为 -1

### 参考代码

```python
from collections import deque

def shortest_path(maze, N, M):
    # 找起点和终点
    start = end = None
    for i in range(N):
        for j in range(M):
            if maze[i][j] == 'S':
                start = (i, j)
            elif maze[i][j] == 'E':
                end = (i, j)
    
    if not start or not end:
        return -1
    
    # 方向数组
    dirs = [(-1,0), (1,0), (0,-1), (0,1)]
    
    # 距离数组: dist[x][y][key_mask]
    # 用字典存储以节省空间 (最多 10000 * 1024 约 10^7，可能太大)
    # 使用三维列表，但用 -1 初始化
    dist = [[[-1] * (1 << 10) for _ in range(M)] for _ in range(N)]
    
    sx, sy = start
    dist[sx][sy][0] = 0
    q = deque()
    q.append((sx, sy, 0))
    
    while q:
        x, y, mask = q.popleft()
        d = dist[x][y][mask]
        
        # 到达终点
        if (x, y) == end:
            return d
        
        for dx, dy in dirs:
            nx, ny = x + dx, y + dy
            
            # 边界检查
            if nx < 0 or nx >= N or ny < 0 or ny >= M:
                continue
            
            cell = maze[nx][ny]
            
            # 障碍物
            if cell == '1':
                continue
            
            new_mask = mask
            
            # 遇到钥匙 (大写字母 A-J)
            if 'A' <= cell <= 'J':
                bit = ord(cell) - ord('A')
                new_mask = mask | (1 << bit)
            
            # 遇到锁 (小写字母 a-j)
            if 'a' <= cell <= 'j':
                bit = ord(cell) - ord('a')
                if not (mask & (1 << bit)):
                    continue  # 没有对应钥匙
            
            # 如果这个状态没访问过
            if dist[nx][ny][new_mask] == -1:
                dist[nx][ny][new_mask] = d + 1
                q.append((nx, ny, new_mask))
    
    return -1


if __name__ == "__main__":
    N, M = map(int, input().split())
    maze = [list(input().strip()) for _ in range(N)]
    print(shortest_path(maze, N, M))
```
