## Problem 1: 简易计算器 (100分)

### 题目描述

实现一个简易计算器，支持四则运算（`+`, `-`, `*`, `/`）和括号 `()`。给定一个合法的数学表达式字符串，请计算其结果。

**注意：**
- 表达式中的数字均为非负整数（可能为多位数）
- 除法运算向零取整（`int(a / b)`），题目保证除数不为0
- 表达式长度不超过 100,000 字符

### 输入格式

一行字符串，表示待计算的数学表达式。

### 输出格式

一个整数，表示计算结果。

### 样例

```
输入: "1+2*3"
输出: 7

输入: "(3+5)*2"
输出: 16

输入: "10-4/2"
输出: 8

输入: "2*(3+4*(5-2))"
输出: 30
```

### 数据范围

| 参数 | 范围 |
|------|------|
| 表达式长度 | 1 ≤ len ≤ 10⁵ |
| 数字大小 | 0 ≤ num ≤ 10⁹ |
| 中间结果 | 保证运算过程中不超出64位整数范围 |

### 解题思路

这是一个经典的中缀表达式求值问题，可使用**双栈法**（操作数栈 + 运算符栈）：

1. **遍历字符串**，遇到数字则解析完整数字并入操作数栈
2. 遇到 `(` 直接入运算符栈
3. 遇到 `)` 弹出运算符直到遇到 `(`
4. 遇到 `+`, `-`, `*`, `/`：当前运算符优先级 ≤ 栈顶优先级时，先计算栈顶
5. **优先级**：`*`, `/` 为 2 级；`+`, `-` 为 1 级

**简化思路**（两遍扫描）：
- 第一遍处理 `*` 和 `/`，将中间结果放回
- 第二遍处理 `+` 和 `-`

### 参考代码

```python
def calculate(s):
    # 使用栈的方式处理
    # 方法：将所有运算转化为加法
    stack = []
    num = 0
    sign = '+'  # 当前数字前的运算符
    i = 0
    n = len(s)
    
    while i < n:
        c = s[i]
        if c.isdigit():
            num = num * 10 + int(c)
        elif c == '(':
            # 找到匹配的括号，递归计算内部表达式
            cnt = 1
            j = i + 1
            while j < n and cnt > 0:
                if s[j] == '(':
                    cnt += 1
                elif s[j] == ')':
                    cnt -= 1
                j += 1
            num = calculate(s[i+1:j-1])
            i = j - 1  # 跳过已处理的部分
        if c in '+-*/)' or i == n - 1:
            if c == ')' and sign == '+':
                # do nothing, sign already set
                pass
            elif sign == '+':
                stack.append(num)
            elif sign == '-':
                stack.append(-num)
            elif sign == '*':
                stack.append(stack.pop() * num)
            elif sign == '/':
                left = stack.pop()
                # 向零取整
                stack.append(int(left / num))
            num = 0
            sign = c
        i += 1
    
    return sum(stack)


# 主程序
if __name__ == "__main__":
    expr = input().strip()
    print(calculate(expr))
```

## Problem 2: 最大连续子序列和 (100分)

### 题目描述

给定一个整数数组 `nums`，请找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。同时返回该子数组的起始和结束索引（索引从0开始）。

如果有多个子数组具有相同的最大和，返回**起始索引最小**的那个；若仍有多个，返回**结束索引最小**的那个。

### 输入格式

第一行：一个整数 `n`，表示数组长度（1 ≤ n ≤ 10⁵）  
第二行：`n` 个整数，用空格分隔

### 输出格式

一行，三个整数 `max_sum start end`，用空格分隔。

### 样例

```
输入:
9
-2 1 -3 4 -1 2 1 -5 4

输出:
6 3 6

解释: 最大和为 4 + (-1) + 2 + 1 = 6，起始索引3，结束索引6
```

```
输入:
5
-1 -2 -3 -4 -5

输出:
-1 0 0

解释: 全为负数，最大值是 -1, 索引[0,0]
```

### 数据范围

| 参数 | 范围 |
|------|------|
| n | 1 ≤ n ≤ 10⁵ |
| nums[i] | -10⁴ ≤ nums[i] ≤ 10⁴ |

### 解题思路

**Kadane算法**（动态规划）O(n) 时间，O(1) 空间：

- 定义 `cur_max` 表示以当前位置结尾的最大子数组和
- 定义 `global_max` 表示全局最大子数组和
- 递推：`cur_max = max(num, cur_max + num)`
- 当 `cur_max` 重置为 `num` 时，更新起始索引
- 当 `cur_max > global_max` 或 `cur_max == global_max` 且满足索引条件时更新结果

### 参考代码

```python
def max_subarray(nums):
    n = len(nums)
    if n == 0:
        return 0, 0, 0
    
    cur_max = nums[0]
    global_max = nums[0]
    start = 0
    end = 0
    cur_start = 0
    
    for i in range(1, n):
        if cur_max + nums[i] < nums[i]:
            cur_start = i
            cur_max = nums[i]
        else:
            cur_max = cur_max + nums[i]
        
        if cur_max > global_max:
            global_max = cur_max
            start = cur_start
            end = i
        elif cur_max == global_max:
            # 相同最大和，选起始索引最小的
            if cur_start < start or (cur_start == start and i < end):
                start = cur_start
                end = i
    
    return global_max, start, end


if __name__ == "__main__":
    n = int(input().strip())
    nums = list(map(int, input().strip().split()))
    max_sum, start_idx, end_idx = max_subarray(nums)
    print(max_sum, start_idx, end_idx)
```
