## 1. 解码字符串 ✅
**Leetcode 394 · L75**

**问题**：`"3[a2[c]]"` → `"accaccacc"`。`k[encoded]` 表示 encoded 重复 k 次。

**思路**：两个栈分别存数字和前面的字符串。遍历时：数字入数字栈；字母累加；`[` 把当前数字和已拼字符串入栈（作为上下文保存）；`]` 弹出数字和前面的字符串拼在一起。

```python
def decodeString(s: str) -> str:
    num_stack = []
    str_stack = []
    cur_num = 0
    cur_str = ''
    for ch in s:
        if ch.isdigit():
            cur_num = cur_num * 10 + int(ch)
        elif ch == '[':
            num_stack.append(cur_num)
            str_stack.append(cur_str)
            cur_num, cur_str = 0, ''
        elif ch == ']':
            cur_str = str_stack.pop() + cur_str * num_stack.pop()
        else:
            cur_str += ch
    return cur_str
```

**易错点**：数字可能多位（`12[a]`）；`]` 后拼 string 的顺序：`prev + cur * k` 而非 `cur * k`。

**变种**：394 也可以用递归（解析子串）+栈结合。

## 2. 每日温度 ✅
**Leetcode 739 · L75/T150/O**

**问题**：给定温度数组，返回每个温度要等多少天才能遇到更高的温度，没有则为 0。

**思路**：单调递减栈。栈存下标，遇到比栈顶大的温度时出栈并计算结果。

```python
def dailyTemperatures(temperatures: list[int]) -> list[int]:
    n = len(temperatures)
    res = [0] * n
    stack = []   # 存下标，栈顶到栈底递增（严格说栈内温度单调递减）
    for i in range(n):
        while stack and temperatures[i] > temperatures[stack[-1]]:
            idx = stack.pop()
            res[idx] = i - idx
        stack.append(i)
    return res
```

**易错点**：栈存下标而非值；while 条件比较的是温度而非下标；记得初始化 `res=[0]*n`。

**变种**：下一个更大元素 I（496）、下一个更大元素 II（503）、接雨水（42）。

## 3. 柱状图中最大的矩形 ✅
**Leetcode 84 · O**

**问题**：给定 n 个非负整数（柱高），求能勾勒出的最大矩形面积。

**思路**：单调递增栈（存下标）。遍历时遇到更矮的柱子则弹出栈顶，以弹出高度为高，左右边界为当前下标和栈顶下标计算面积。最后清空栈。

```python
def largestRectangleArea(heights: list[int]) -> int:
    stack = []
    max_area = 0
    # 末尾加 0 保证最后清空栈
    heights = heights + [0]
    for i in range(len(heights)):
        while stack and heights[i] < heights[stack[-1]]:
            h = heights[stack.pop()]
            w = i if not stack else i - stack[-1] - 1
            max_area = max(max_area, h * w)
        stack.append(i)
    return max_area
```

**易错点**：宽度计算 `i - stack[-1] - 1`（以弹出高度为高，左右第一个更矮的柱子为边界）；末尾加哨兵 0；栈存下标。

**变种**：最大矩形（85）、接雨水（42，单调递减栈）。

## 4. 最大矩形 ✅
**Leetcode 85 · O**

**问题**：给定由 '0' 和 '1' 组成的矩阵，找出只包含 '1' 的最大矩形面积。

**思路**：逐行处理，将每一行看作柱状图（当前行及以上连续的 '1' 视为高度），调用 84 题的最大矩形函数求解。

```python
def maximalRectangle(matrix: list[list[str]]) -> int:
    if not matrix or not matrix[0]: return 0
    m, n = len(matrix), len(matrix[0])
    heights = [0] * n
    max_area = 0
    for i in range(m):
        for j in range(n):
            heights[j] = heights[j] + 1 if matrix[i][j] == '1' else 0
        max_area = max(max_area, largestRectangleArea(heights))
    return max_area
```

**易错点**：高度累加（断续为 0）；字符 '1' 而非数字 1；可复用 84 的函数。

**变种**：矩阵中的正方形（221）——用 DP 而非单调栈。

## 5. 基本计算器 ✅
**Leetcode 224 · T150**

**问题**：实现包含 `+ - ( )` 和空格的整数计算器，可正可负。

**思路**：栈存符号和结果。当前符号 `sign`（±1），遇到数字累加，遇到 `+/-` 更新 sign，遇到 `(` 把当前结果和符号入栈然后重置，遇到 `)` 弹出之前的结果和符号进行计算。

```python
def calculate(s: str) -> int:
    stack = []
    res, sign, i = 0, 1, 0
    while i < len(s):
        ch = s[i]
        if ch.isdigit():
            num = 0
            while i < len(s) and s[i].isdigit():
                num = num * 10 + int(s[i])
                i += 1
            res += sign * num
            continue
        elif ch == '+': sign = 1
        elif ch == '-': sign = -1
        elif ch == '(':
            stack.append(res)
            stack.append(sign)
            res, sign = 0, 1
        elif ch == ')':
            res = stack.pop() * res + stack.pop()
        i += 1
    return res
```

**易错点**：`res` 和 `sign` 在遇到 `(` 时入栈顺序（先 res 后 sign，弹出时先 sign 后 res）；数字可能有多位；`i` 的自增需小心。

## 6. 基本计算器 II ✅
**Leetcode 227 · T150**

**问题**：包含 `+ - * /` 和空格的整数计算器，无括号。

**思路**：一次遍历，当前数字和运算符。遇到 `+` 把数字压栈（正数），`-` 压负数，`*` 和 `/` 将栈顶出栈与当前数字运算后重新入栈。最后求和。

```python
def calculate(s: str) -> int:
    stack = []
    num, op = 0, '+'
    s = s.replace(' ', '')
    for i, ch in enumerate(s):
        if ch.isdigit():
            num = num * 10 + int(ch)
        if ch in '+-*/' or i == len(s) - 1:
            if op == '+': stack.append(num)
            elif op == '-': stack.append(-num)
            elif op == '*': stack.append(stack.pop() * num)
            elif op == '/': stack.append(int(stack.pop() / num))
            op = ch
            num = 0
    return sum(stack)
```

**易错点**：除法向零取整 `int(a/b)`；最后一次无符号时也要运算（`i == len(s)-1`）；空格先去除。

**变种**：表达式求值综合题（770. Basic Calculator IV 更复杂）。

|------|---------|--------|
| 解码字符串 | 双栈（数字+字符串），遇到 `[` 保存上下文 | 辅助栈 |
